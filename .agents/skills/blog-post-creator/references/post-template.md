# Markdown Post Template

Use this template for new files under `content/posts/<slug>.md`.

```markdown
---
title: "Post Title"
date: "YYYY-MM-DD"
excerpt: "Single-sentence concrete takeaway."
coverImage: "/assets/blog/<slug>/cover.webp"
tags:
  - "Tag One"
  - "Tag Two"
---

Opening paragraph: establish context and thesis in 2-4 sentences.

## Section Heading One

Develop one core claim with mechanism or example.

## Section Heading Two

Explain decisions, trade-offs, or implications.

## Section Heading Three (optional)

Add supporting detail only when it advances the thesis.

Closing paragraph: synthesize the argument and end on a concrete forward point.
```

## Frontmatter Notes

- Include `updated` only when materially revising an existing post.
- Keep `excerpt` specific and outcome-oriented.
- Use 2-4 tags; prefer existing site tag language when possible.
- Keep `coverImage` path aligned to slug directory naming.
